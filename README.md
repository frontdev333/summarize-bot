# Summarize Bot

## Описание проекта
Telegram-бот для автоматического сбора новостей по подпискам, их краткого изложения (суммаризации) с помощью нейросетей и рассылки пользователям в Telegram. Позволяет оперативно получать ключевые выжимки из различных новостных источников.

## Возможности
- Подписка на новостные источники и тематики.
- Получение коротких выжимок (summary) из длинных новостных статей.
- Автоматическая рассылка агрегированных новостей пользователям в Telegram.

## Требования
- Go 1.21+
- Токен Telegram бота (выдается в @BotFather)
- Ключ от NewsAPI (или другого новостного провайдера)
- Ключ от Google Gemini (или другого ИИ для суммаризации)

## Переменные окружения (Конфигурация)
| Параметр | Описание | Значение по умолчанию |
|---|---|---|
| `bot_token` | Токен Telegram-бота | - |
| `news_api_key` | Ключ доступа к NewsAPI | - |
| `gemini_api_key` | Ключ доступа к API Gemini (Google AI) | - |
| `poll_interval` | Интервал проверки и рассылки новостей | `60m` |

## Инструкции по запуску
1. Клонируйте репозиторий:
   ```bash
   git clone <url_репозитория>
   cd summarize-bot
   ```
2. Скопируйте пример конфига и заполните его:
   ```bash
   cp config.example.yaml config.yaml
   ```
3. Установите зависимости модулей:
   ```bash
   go mod tidy
   ```
4. Запустите бота:
   ```bash
   go run cmd/cli/main.go
   ```

## Структура проекта
- `cmd/cli/` — Точка входа в приложение (`main.go`).
- `internal/` — Внутренняя бизнес-логика:
  - `app/` — Сборка зависимостей приложения.
  - `cache/` — Кэширование данных.
  - `config/` — Парсинг и управление настройками приложения.
  - `news/` — Взаимодействие с внешними новостными API.
  - `subscriptions/` — Логика управления подписками пользователей бота.
  - `summary/` — Сервис создания текстовой выжимки с использованием ИИ.
  - `telegram/` — Обработчики Telegram-бота (команды, сообщения).

## Используемые технологии
- **Язык**: Go (1.21+)
- **Telegram API**: Интеграция с Telegram
- **News API**: Источник свежих новостей
- **Gemini (Google AI)**: Генерация коротких саммари (выжимок)
